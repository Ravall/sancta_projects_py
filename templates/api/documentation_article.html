Получение статей

Статьи можно получить (как и события) по id: {{ article_by_id_example }},
и по его name: {{ article_by_name_example }}

Пример ответа:
{
    "name": "kak_pitatsya_v_velikiy_post",
    "tags": [
        "post"
    ],
    "text": {
        "content": "content",
        "annonce": "annonce",
        "title": "title"
    },
    "image": "image",
    "type": "article",
    "id": 228
}
Пример ответа со статьей привязанной к событию:
{
    "text": {
        "content": "content",
        "annonce": "annonce",
        "title": "title"
    },
    "image": "",
    "type": "article",
    "id": 231,
    "related": [
        {
            "image": "1323196997_738.jpg",
            "type": "event",
            "id": 7,
            "name": "pasha"
        }
    ]
}
Cледует обратить внимание!
  1. Секция tags может отсутствовать
  2. Секция related показывает объект к которому привязана статья
  3. Порядок полей ответа может быть иным

Статьи можно получать по тегам: {{ article_by_tag_example }}. При этом список секций будет сокращенных 

Пример ответа:
[
    {
        "image": "",
        "type": "article",
        "id": 228,
        "name": "kak_pitatsya_v_velikiy_post",
        "tags": [
            "post"
        ]
    }
]
