Общая информация
  1. обращение к API осуществляется через GET запрос на нужный урл
  2. Ответ API возможен в двух форматах 'json' и 'xml'. Формат задается суфиком запроса: http://*api_request*.[xml|json].
  3. Некоторые секции ответов могут отсутствовать или возвращены пустым значением
  4. Чтобы получить изображение, имя которого указано в секции images, нужно запросить его по адресу http://img.sancta.ru/crop/image_size/image_name. В image_size необходимо подставить нужный размер из доступных - "150x200", "600x800"
