Православный календарь

Можно получить список православных событий на нужный день календаря. API содержит {{ events_count }} православных праздников.

База праздников постоянно прорабатывается - улучшается содержание, описание, добавляются новые праздники, дни святых, православные седмицы.

Иконы

К каждому событию, по возможности подгружаются иконы. Иконы доступны в нескольких разрешениях.

Статьи

Статьи тематические, привязанные к определенным событиями или сгруппированные между собой по определенной тематике, дополняют информационную базу календаря. Статьи написаны заинтересованными пользователями, прошедшие вычитку и коррекцию.

Вы можете прислать нам свою авторскую статью - если она удовлетворит нашим правилам, то мы ее опубликуем.
